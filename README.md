# Electronics Lab

Python and SCPI scripts for controlling laboratory instruments:

- Siglent SDS1204X-E oscilloscope

- Siglent function generator

- Siglent laboratory power supply

- Measurement automation for RC, RL, and RLC circuits

## Goal

Automated measurements, reproducible experiments, and later use for laboratory reports and book projects.

# Elektroniklabor

- Siglent SDS1204X-E Oszilloskop
- Siglent Funktionsgenerator
